# **App Name**: LitreLink

## Core Features:

- Kilogram Input: A dedicated input field for users to enter a numerical value in kilograms.
- Litre Output Display: Displays the calculated equivalent volume in litres in a clear, read-only format.
- Real-time Conversion Engine: Automatically converts the input kilogram value to litres using the specified rate of 1 kg = 0.97 Litre, updating in real-time.
- Conversion Rate Indicator: Clearly displays the fixed conversion rate (1 kg = 0.97 Litre) on the interface for user reference.

## Style Guidelines:

- Primary color: A professional and calming blue (#336699), chosen to convey reliability and precision.
- Background color: A subtle and very light shade of the primary blue (#ECF1F5), ensuring clarity and focus for the main content.
- Accent color: A vibrant, clear blue-green (#36AFDB) that is analogous to the primary color, providing good contrast for interactive elements or highlights.
- All text will use 'Inter', a grotesque-style sans-serif font known for its modern, objective, and neutral aesthetic, ensuring high readability for numerical data and labels.
- Utilize simple and intuitive icons to represent input (kilograms) and output (litres), reinforcing the conversion process without visual clutter.
- A clean, focused layout with the input and output fields prominently displayed in a central card-like component, maximizing readability and minimizing distractions. Ample whitespace will frame the core functionality.
- Subtle and functional animations will be used for state changes, such as when conversion results update or input fields are focused, providing gentle user feedback without being obtrusive.
